# calculator-project

A simple calculator application written in Python that performs basic arithmetic operations.

## Overview
This project is a beginner-friendly Python calculator designed to demonstrate the use of:
- Variables
- User input
- Conditional statements
- Functions
- Basic error handling

It can be used as a simple portfolio project for showcasing Python fundamentals.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly command-line interaction
- Basic input validation
- Division by zero handling

## Technologies Used
- Python 3

How to Run

    Make sure Python 3 is installed on your system.
    Open a terminal in the project folder.
    Run the following command:

    Enter the first number: 10
    Enter the second number: 5
    Choose operation (+, -, *, /): *
